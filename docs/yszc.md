# 隐私政策

**本app仅用于学习和个人使用，我们承诺不滥用您的个人信息。**

## 1. 我们收集的信息
为保障基础学习功能正常运行，我们会收集以下信息：

| 信息类别 | 具体内容 | 收集目的 |
| :--- | :--- | :--- |
| **账号信息** | 手机号 / 邮箱、昵称 | 注册、登录、找回账号 |
| **学习记录** | 课程进度、练习结果、完成时间 | 记录学习成果、生成学习报告 |

## 2. 信息使用
- 仅用于提供学习功能、保存个人进度和优化体验。
- **我们不会**将您的信息出售、出租或用于任何商业推广。
- 本app不接入第三方广告SDK，不进行用户画像商业分析。

## 3. 信息共享
仅在以下极少数情况下共享信息：
- 法律要求（如司法机关依法调取）
- 为发送短信验证码而使用第三方通道（仅传输手机号）

除此之外，**我们不会主动将您的个人信息共享给任何其他第三方**。

## 4. 数据存储与安全
- 我们采用常规加密和安全措施保护您的数据。
- 您注销账号后，相关学习数据将在[例如：30天]内删除。

## 5. 您的权利
您可以：
- 在app内修改昵称等个人资料
- 查看自己的学习记录
- 联系客服注销账号并删除数据

## 6. 未成年人
- 若您未满14周岁，请在监护人同意下使用本app。
- 我们不会主动收集未成年人的非必要信息。

## 7. 隐私政策更新
如有重大变更，我们会通过app内公告通知您。
